draw

탭해서 핵심 뉘앙스 보기 →
끌어내다

탭해서 단어들 보기 →
'끌어내다'로 모든 뜻이 풀립니다
그림을 그리다
붓으로 선을 끌어내다
총을 뽑다
홀스터에서 끌어내다
커튼을 치다
옆으로 끌어내다
추첨을 하다
상자에서 끌어내다
활 시위를 당기다
뒤로 끌어내다
주의를 끌다
시선을 끌어내다

Latin 'portare'
port
항구에서 컨테이너를 배에서 육지로 나르는 장면
탭해서 핵심 뉘앙스 보기 →
나르다
port의 단 하나의 본질
탭해서 단어들 보기 →
'나르다'로 모든 뜻이 풀립니다
import
im(안으로) → 수입하다
export
ex(밖으로) → 수출하다
transport
trans(건너서) → 운송하다
report
re(다시) → 보고하다
support
sub(아래서) → 지지하다
portable
able → 휴대용의

Latin 'videre / visus'
vis / vid
전망대에서 망원경으로 도시 전체를 바라보는 장면
탭해서 핵심 뉘앙스 보기 →
보다
vis의 단 하나의 본질
탭해서 단어들 보기 →
'보다'로 모든 뜻이 풀립니다
vision
보는 능력 → 시각
visible
볼 수 있는
visual
보는 것의 → 시각적인
supervise
위에서 보다
provide

television
멀리 보는 것 → TV

왜 그림으로 배워야 하는가

🧠
6.5배
사진이 텍스트보다 오래 기억된다
3일 후 기억 보존율: 텍스트 10% vs 사진 65%
— John Medina, Brain Rules (2008)
🔗
2배
뇌의 기억 경로가 두 배로 활성화된다
그림과 언어를 함께 학습하면 시각 + 언어 경로가 동시에 저장된다
— Allan Paivio, Dual Coding Theory (1971)
💡
1개
핵심 뉘앙스 하나로 수십 개 뜻이 풀린다
모든 단어의 뜻은 하나의 핵심 이미지에서 파생된다
— Eleanor Rosch, Prototype Theory (1970s)
